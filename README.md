# minas-valor

O projeto fornece uma landing page para a cooperativa bancária Minas Valor usando apenas html e css que será renderizados pelo nginx da virtual machine EC2 AWS configurada durante o projeto.

## Objetivo

O projeto de Infraestrutura de Redes do Eixo 5 de Sistemas de Informação (PUC Minas) tem por objetivo planejar, prototipar, documentar e implementar uma infraestrutura de rede para uma Cooperativa bancária fictícia, chamada de Minas Valor, em uma cidade polo do interior do estado de Minas Gerais, em Poços de Caldas, com 5 filiais em cidades próximas, Águas da Prata, São João da Boa Vista, Bandeira do Sul, São José do Rio Pardo e Andradas.
